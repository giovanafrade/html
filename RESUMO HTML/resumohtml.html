HTML é uma linguagem de marcação que define o significado, estrutura e o conteúdo de páginas da web, sendo o componente base da web.
Os elementos HTML são separados de outro texto em um documento por "tags", que consistem no nome do elemento entre "<" e ">", Essas tags
ajudam a definir a estrutura e a função do conteúdo de uma página web, facilitando a leitura e interpretação tanto para humanos quanto para
máquinas, como navegadores, motores de busca e tecnologias assistivas. Podemos afirmar também que o HTML não é "Case Sensitive", isto é,
pode ser escrito em maiúsculas, minúsculas ou uma mistura. Vamos iniciar nosso estudo com essa linguagem?


Estrutura básica do HTML

O documento HTML sempre inicia com o que chamamos de estrutura básica. Esta estrutura é quase que imutável. Sempre será dessa forma que você
começará seu HTML.

➤ < html >: Instrução para o navegador e outros programas que podem ler o site, que o código encontrado ali é um código HTML. Assim eles sabem
o que fazer para mostrar seu site da melhor forma possível.
➤ < head >: Contém informações que não são transpostas visivelmente para o usuário/leitor do documento. São dados implícitos, de uso e controle do
documento: vinculação com outros arquivos, aplicação de lógica de programação de scripts e metadados. Na prática, todo o conteúdo do cabeçalho fica
demilitado entre a abertura e fechamento tag head.
➤ < body >: Trata-se de toda informação legível para o usuário/leitor do documento. É todo e qualquer texto que se deseja apresentar, assim como
toda e qualquer forma de mídia de saída (imagens, sons, miniaplicativos embutidos, conteúdo multimídia, etc). Além disso, toda a apresentação de
entrada de dados (formulários) também se aplica neste seção do documento.

Vantagens da utilização do HTML

  1. Acessibilidade: O HTML oferece recursos para melhorar a acessibilidade
    das páginas da web, permitindo que pessoas com deficiências visuais ou motoras possam
    utilizar softwares leitores de tela e navegar pelo conteúdo. A utilização de tags adequadas,
    texto alternativo para imagens e atributos de acessibilidade contribuem para tornar seu site
    mais acessível para todos os usuários.

  2. Estruturação de conteúdo: Com o HTML, é possível estruturar o
    conteúdo da página de forma clara e organizada, facilitando a leitura e
    compreensão dos usuários. Os elementos HTML, como cabeçalhos, parágrafos,
    listas e tabelas, permitem uma organização hierárquica e lógica do conteúdo.

  3. Integração com outras tecnologias: O HTML pode ser facilmente
    combinado com outras tecnologias, como CSS para estilização
    e layout, JavaScript para interatividade e funcionalidades
    avançadas, e APIs para integração com serviços externos. Essa
    flexibilidade de integração permite criar páginas web ricas e dinâmicas.

  4. Compatibilidade: O HTML é suportado por todos os principais
    navegadores da web, o que significa que seu site terá uma
    ampla compatibilidade e poderá ser visualizado corretamente
    em diferentes dispositivos e plataformas.
